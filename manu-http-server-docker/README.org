#+TITLE: Manu Http Server Docker
* ¿Qué es?
  - Servidor http
* ¿Para qué sirve?
  - Ejecutar paquetes npm del proyecto inicializado
  - Evitar instalar paquetes npm de forma global con -g
* Requisitos
  - Requiere tener instalado las siguientes tecnologías
    1) npm
    2) nodejs
    3) docker
    4) docker-compose
